How to Use These API Keys

Add these keys to your .env file in the PunchAI MCP Server directory:

API_KEYS=your-key-1,your-key-2,your-key-3

When connecting from a client, include the API key in the WebSocket connection headers:

const ws = new WebSocket('ws://your-server:3000/mcp', {
  headers: {
    'X-API-Key': 'your-api-key-here'
  }
});
